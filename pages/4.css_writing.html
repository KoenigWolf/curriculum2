CSS 記述

1. CSSとは？

CSS（Cascading Style Sheets）は、HTMLで作成されたウェブページの見た目を装飾するためのスタイルシート言語です。CSSを使うことで、テキストの色やフォント、レイアウトなどを指定し、ウェブページを美しく見やすく整えることができます。

2. CSSの基本的な記述方法

CSSは、セレクタ（どの要素にスタイルを適用するかを指定する部分）と、そのスタイルを定義するプロパティと値で構成されます。以下が基本的なCSSの記述例です。

            /* h1要素のテキストカラーを青にする */
            h1 {
            color: blue;
            }

            /* p要素のフォントサイズを16pxに設定する */
            p {
            font-size: 16px;
            }

上記のように、セレクタで指定したHTML要素に対して、プロパティとその値を中括弧 `{}` の中に記述します。

3. CSSの適用方法

CSSをHTMLに適用する方法はいくつかありますが、ここでは3つの基本的な方法を紹介します。

  • インラインスタイル: HTMLタグ内に直接記述する方法です。 
    <p style="color: red;">このテキストは赤色です。</p>
  • 内部スタイルシート: HTMLファイルの`<head>`タグ内に`<style>`タグを使って記述します。 
                      <style>
                  p {
                  color: green;
                  }
                  </style>
  • 外部スタイルシート: CSSを外部ファイルに記述し、それをHTMLファイルにリンクする方法です。最も推奨される方法です。 
                      /* styles.css */
                  p {
                  color: blue;
                  }

                  
                  <link rel="stylesheet" href="styles.css">

外部スタイルシートを使うと、複数のHTMLファイルに対して一貫したスタイルを適用できます。

4. CSSの基本プロパティ

CSSには多くのプロパティがありますが、以下はよく使われる基本的なプロパティです。

  • color: テキストの色を指定します。
  • background-color: 背景色を指定します。
  • font-size: テキストのサイズを指定します。
  • margin: 要素の外側の余白を指定します。
  • padding: 要素の内側の余白を指定します。
  • border: 要素の枠線を指定します。

5. CSSのセレクタ

CSSのセレクタには、要素セレクタ、クラスセレクタ、IDセレクタなどがあります。それぞれの使い方を以下に示します。

  • 要素セレクタ: HTMLの要素名を使ってスタイルを適用します。 
    h1 { color: red; }
  • クラスセレクタ: クラス属性に基づいてスタイルを適用します。クラス名の前にドット `.` をつけます。 
    .highlight { color: yellow; }
  • IDセレクタ: ID属性に基づいてスタイルを適用します。ID名の前にハッシュ `#` をつけます。 
    #header { background-color: grey; }
