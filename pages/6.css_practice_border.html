CSS 練習⓵(Border)

1. CSSのborderプロパティとは？

CSSのborderプロパティは、要素の周りに枠線を追加するためのプロパティです。borderは、色、太さ、スタイルの3つの要素で構成されており、これらを組み合わせて枠線をカスタマイズできます。

2. 基本的なborderの使い方

以下のコード例では、borderプロパティを使用して、要素の周りに1pxの黒い実線を設定しています。

            /* p要素に1pxの黒い実線を適用 */
            p {
            border: 1px solid black;
            }

上記の例では、1pxが線の太さ、solidが線のスタイル、blackが線の色を指定しています。

3. borderのカスタマイズ

borderプロパティを使って、線の太さ、スタイル、色を自由にカスタマイズすることができます。以下の例を見てみましょう。

            /* div要素にカスタマイズされた枠線を適用 */
            div {
            border-width: 3px;
            border-style: dashed;
            border-color: blue;
            }

この例では、border-widthで線の太さを3pxに、border-styleで線のスタイルを破線（dashed）、border-colorで線の色を青に設定しています。

4. 個別のborderプロパティ

borderプロパティは、要素の四辺すべてに同じスタイルを適用しますが、各辺ごとに異なるスタイルを適用することも可能です。以下のプロパティを使って、個別に設定できます。

  • border-top: 上辺の枠線を指定
  • border-right: 右辺の枠線を指定
  • border-bottom: 下辺の枠線を指定
  • border-left: 左辺の枠線を指定

例：

            /* div要素の各辺に異なる枠線を適用 */
            div {
            border-top: 2px solid red;
            border-right: 5px dotted green;
            border-bottom: 3px dashed blue;
            border-left: 4px double black;
            }

5. 練習課題: 枠線を使ったデザインの作成

以下の課題に挑戦してみましょう。簡単なボックスデザインを作成し、borderプロパティを使ってスタイルをカスタマイズしてください。

  • 1. div要素を作成し、その周りに赤い実線の枠線（2px）を追加します。
  • 2. p要素を作成し、その周りに青い破線の枠線（3px）を追加します。
  • 3. 上記の枠線スタイルを応用して、独自のデザインを作成してみてください。
