CSS 練習⓶(Class名付与)

1. クラス名とは？

クラス名は、HTML要素にスタイルを適用するためのラベルのようなものです。複数の要素に同じクラス名を付与することで、同じスタイルをまとめて適用することができます。これにより、効率的にデザインを管理することができます。

2. クラス名の付与方法

HTML要素にクラス名を付与するには、class属性を使用します。以下は、div要素にクラス名を付与する例です。

            <div class="container">
            ここにコンテンツが入ります。
            </div>

この例では、class="container"と指定することで、このdiv要素に「container」というクラス名が付与されています。

3. CSSでクラス名を使ったスタイル適用

クラス名を使ってスタイルを適用するには、CSSファイル内でクラス名の前にドット（.）を付けて指定します。以下は、先ほどの「container」クラスにスタイルを適用する例です。

            /* containerクラスに対してスタイルを適用 */
            .container {
            width: 80%;
            margin: 0 auto;
            padding: 20px;
            background-color: #f4f4f4;
            }

この例では、「container」クラスに対して、幅（width）、マージン（margin）、パディング（padding）、背景色（background-color）が適用されています。

4. 複数のクラス名を適用する

1つのHTML要素に対して、複数のクラス名を付与することも可能です。クラス名をスペースで区切って指定します。以下の例を見てみましょう。

            <div class="container highlight">
            ここにコンテンツが入ります。
            </div>

この例では、「container」と「highlight」という2つのクラス名が付与されています。それぞれのクラスにCSSでスタイルを定義することで、要素に複数のスタイルを適用できます。

            /* containerクラスのスタイル */
            .container {
            width: 80%;
            margin: 0 auto;
            padding: 20px;
            background-color: #f4f4f4;
            }

            /* highlightクラスのスタイル */
            .highlight {
            border: 2px solid red;
            }

このように、.containerと.highlightのスタイルが組み合わさって適用されます。

5. 練習課題: クラス名を使ったスタイルの管理

以下の課題に挑戦して、クラス名を使ったスタイルの管理方法に慣れてください。

  • 1. 「box」というクラス名を持つdiv要素を作成し、幅、マージン、背景色を指定します。
  • 2. 「highlight」というクラスを作成し、枠線やフォントサイズを指定します。
  • 3. 「box」と「highlight」の両方を持つdiv要素を作成し、両方のスタイルが適用されていることを確認します。
  • 4. 複数の要素に「highlight」クラスを適用し、デザインの一貫性を保つ方法を試してみてください。

ブラウザで結果を確認し、クラス名を使ったスタイルの適用方法を理解しましょう。
