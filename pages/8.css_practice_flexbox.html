CSS 練習⓷(Flexbox)

1. Flexboxとは？

Flexbox（フレックスボックス）は、CSSのレイアウトモジュールの一つで、要素を柔軟かつ効率的に配置するために使用されます。Flexboxを使用すると、複雑なレイアウトを簡単に作成でき、特にレスポンシブデザインにおいて強力です。

2. Flexboxの基本的な使い方

Flexboxを使用するには、親要素にdisplay: flex;を設定します。これにより、その子要素がフレックスコンテナ内で並べられます。以下は基本的な例です。

            /* 親要素にdisplay: flex;を適用 */
            .container {
            display: flex;
            }

            /* 子要素に対するスタイル（任意） */
            .container div {
            padding: 10px;
            background-color: lightgray;
            margin: 5px;
            }

上記のスタイルを適用すると、子要素が横並びになり、指定されたパディングとマージンが適用されます。

3. フレックスコンテナの主軸と交差軸

Flexboxのレイアウトでは、主軸（main axis）と交差軸（cross axis）という概念が重要です。デフォルトでは、主軸は水平方向（横方向）に設定されますが、flex-directionプロパティを使って変更することができます。

            /* 主軸を縦方向に変更 */
            .container {
            display: flex;
            flex-direction: column;
            }

この例では、flex-direction: column;を指定することで、主軸が縦方向（縦並び）になります。

4. 子要素の配置と並べ方

Flexboxでは、子要素の配置や並べ方を簡単にコントロールできます。主なプロパティとして以下が挙げられます。

  • justify-content: 主軸に沿った配置を指定します（例：左寄せ、中央揃え、右寄せなど）。 
                      /* 子要素を中央に配置 */
                  .container {
                  display: flex;
                  justify-content: center;
                  }
  • align-items: 交差軸に沿った配置を指定します（例：上揃え、中央揃え、下揃えなど）。 
                      /* 子要素を交差軸の中央に揃える */
                  .container {
                  display: flex;
                  align-items: center;
                  }
  • flex-wrap: 子要素がフレックスコンテナ内に収まらない場合に折り返しを設定します。 
                      /* 子要素を折り返す */
                  .container {
                  display: flex;
                  flex-wrap: wrap;
                  }

5. 練習課題: Flexboxを使ったレイアウト作成

以下の課題に挑戦して、Flexboxを使ったレイアウト作成に慣れてください。

  • 1. Flexboxを使用して、横並びの3つのボックスを作成します。それぞれのボックスには、適切なパディングとマージンを設定してください。
  • 2. ボックスを中央揃えにし、ページの中央に配置します。
  • 3. ボックスが画面幅を超える場合に、折り返されるように設定します。
  • 4. 縦並びのレイアウトを作成し、ボックスを上下に中央揃えにします。

ブラウザで結果を確認し、Flexboxを使ったレイアウトの自由度と使いやすさを体感してください。
